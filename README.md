# Customer Churn Prediction using Supervised Learning

Project Overview:

This project analyzes customer churn in a telecom company using supervised machine learning techniques. The goal is to identify key factors that influence churn and build models to predict whether a customer is likely to leave the service.

Dataset:

* Name: Telco Customer Churn Dataset
* Source: Kaggle
* Link: https://www.kaggle.com/datasets/blastchar/telco-customer-churn
* Rows: 7043
* Columns: 21

Objective:

To predict customer churn (Yes/No) using customer demographic, service, and billing information, and to analyze factors affecting churn.

Research Questions:

1. How does customer tenure influence churn probability?
2. What is the relationship between monthly charges and churn?
3. Do contract types affect churn rates?
4. How do services (Internet, Online Security, Tech Support) impact churn?
5. How do demographic factors influence churn?

 Methodology:
 
* Data loading and preprocessing
* Handling missing values
* Exploratory Data Analysis
* Visualization of results

Machine Learning Models Used:
* classification
  
Evaluation Metrics:
* Accuracy
* Precision
  
How to Run the Project:

1. Download the dataset from the link above.
2. Open the notebooks in Jupyter Notebook, vscode or Kaggle.
3. Run all cells in each notebook.
4. Outputs (figures and tables) will be saved automatically.

 Outputs:

* Graphs saved as PDF files
* Tables saved as CSV files

Conclusion:

This project identifies key factors influencing customer churn and demonstrates how machine learning models can be used to predict customer behavior effectively.
